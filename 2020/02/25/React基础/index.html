发表于 2020-02-25 | 更新于 2020-02-26
React
大前端 | React

React的特点：

  1. 视图层框架

    • 一个构建用户界面的框架

    • 声明式的框架

    • 数据驱动DOM，再用事件反馈给数据

  2. 组件化

    • 组件组合而不是继承

    • state&&props

    • 生命周期

  3. [^JSX]表达式（处理数据和DOM的关系）

    • 一种[^JS]扩展的表达式
    • 带有逻辑的标记语法，有别于HTML模板
    • 对样式，逻辑表达式和事件的支持

  4. 虚拟DOM

    • 对DOM进行模拟
    • 比较操作前后的数据差异
    • 如果由数据差异，统一操作DOM

React的优点和缺点

优点

  1. 简洁
  2. 灵活
  3. 高效

缺点

  1. 思维转换
  2. 依赖生态
  3. 变动频繁
