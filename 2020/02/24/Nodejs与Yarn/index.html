  2. Yarn的常用语法

    Yarn Npm 备注
    yarn init npm init 项目初始化
    yarn npm install 安装当前文件里package.json里指定文件的包
    yarn global add xxx@x.x.x npm install xxx@x.x.x -g 项目本地安装指定的包
    yarn add xxx@x.x.x npm install xxx@x.x.x –save 项目本地安装指定的包
    yarn add xxx@x.x.x –dev npm install xxx@x.x.x –save-dev 安装一个开发状态下的包
    yarn remove xxx npm uninstall xxx@x.x.x –save(-dev) 删除一个包
    yarn run xxx npm run xxx 执行命令

  1. windows下推荐使用yarn官网安装包安装yarn，也可以使用npm命令下载

  2. 当创建一个项目的时候，首先需要初始化项目，需要用到的命令时yarn init

    注意：windows下不能用git bash初始化，需要使用cmd命令提示符进行yarn init

  3. 初始化后文件夹下会出现一个package.json的文件，存放的是项目相关的包
